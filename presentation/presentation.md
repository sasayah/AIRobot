# 状態 - States

- ジャイロセンサの角度`angle`と角速度`rate`を離散化
  - `0 < angle < 180`を取りうる
  - `70 < angle < 110`を5°ずつ離散化
  - `70 < angle < 110`から出ると失敗
  - `rate`についてはうまく扱えず

---

# 行動 - Actions

- 「進む」、「下がる」、「止まる」の3つ

# Q値

- 状態100 × 行動3 = 300通り
- Q値を保持する配列 : `float[100][3] Q`

---

# Reward

- 前ステップよりも中心の`90°`に近づく
  - 正の`reward`を与える
- 中心から離れる
  - 負の`reward`を与える
